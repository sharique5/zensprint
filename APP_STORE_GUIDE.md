A minimalist, zen-inspired icon that conveys focus, calmness, and precision.

## üöÄ Submission Process

### Apple App Store

1. **Create Apple Developer Account**
   - $99/year fee
   - Visit: developer.apple.com

2. **Prepare in Xcode**
   ```bash
   cd zenSprint
   npx expo prebuild --platform ios
   open ios/zenSprint.xcworkspace
   ```

3. **Configure in App Store Connect**
   - Create new app listing
   - Upload screenshots
   - Add description & metadata
   - Set pricing (Free)
   - Submit for review

4. **Review Timeline**
   - Usually 24-48 hours
   - May request changes if guidelines violated
   - Be responsive to reviewer questions

### Google Play Store

1. **Create Google Play Developer Account**
   - $25 one-time fee
   - Visit: play.google.com/console

2. **Build Android APK/AAB**
   ```bash
   npx expo prebuild --platform android
   cd android
   ./gradlew bundleRelease
   ```

3. **Upload to Play Console**
   - Create new app
   - Upload APK/AAB
   - Add store listing (screenshots, description)
   - Set content rating (ESRB Everyone)
   - Submit for review

4. **Review Timeline**
   - Usually 3-7 days
   - More lenient than Apple
   - Can publish as "Internal Testing" first

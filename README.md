## ğŸ® How to Play

1. Start a session
2. Watch the focus color indicator at the top
3. Tap only circles that match the focus color
4. Avoid tapping wrong colors (penalty)
5. Don't let focus circles fade away (counts as miss)
6. 5 misses = session ends
7. Complete 60 seconds for your final score
